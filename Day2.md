# Values & Data Types
Value is units of information can be in different data type.

## JS has two type of data
1. Primitive data types
    - string
    - number
    - boolean
    - null
    - undefined
      
2. Objects

 ### Coding Examples

```javascript
//typeof return the type of data as a string

//return a string
typeof "hey JS"

//return a number 
typeof 54
```

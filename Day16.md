## Scope
JS organizes scopes with functions and blocks

#### Coding Examples.
```javascript
console.log(null == undefined);
//log true

console.log(42 == [42]); 
//convert [42] ToPrimitive to "42"
//prefer numeric and convert string to 42
//log true

console.log([] == ![]); 
//[] == false 
//"" == false
//0 == false
//0 == 0 
//log true
```
